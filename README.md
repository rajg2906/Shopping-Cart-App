__Shopping Cart Application__

A command-line Java application that simulates a shopping cart system using JDBC to interact with a MySQL database.

__Features__

1. View a catalog of available products.
2. Add items to a shopping cart with real-time stock validation.
3. View the current cart contents and subtotal amount.
4. Process checkouts to deduct funds from a user's digital wallet, replenish inventory if funds are insufficient, and generate a final invoice.
